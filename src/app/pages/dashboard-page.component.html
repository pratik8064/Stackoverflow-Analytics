Analysis and Findings

  • First visualization tells us about popular questions among data collected from all users. This visualization is interactive. On hover one can see count of click action on those question.

    Significance of this visualization is we can see most frequently searched questions. Findings from this visualization is that users are more interested in knowing how to use Swing from JAVA to create user interface, since question on JPanel are frequently searched. Also users are seeking info on HashMap datastructure which means they are not that experianced with Java libraries.

  • Second visualization helps us understand about user's activities. Data collected is properly parsed and showed in table. Viewer can hover on chart to see count of activities and on clicking button can switch between data for current user and total users. doughnut chart shows data for current user and pie chart for all users.

    Significance of this visualization is that we get overview of all activities for users. Findings from this visualization is that users are more prone to user Click and Favorite actiions since they have highest count than other actions. It helps us understand which feature is used most.

  • Third visualization shows us comparison between all user activities. Line chart can easily help us understand relative count of activities. On hover viewer can see count for that activity for respective user. Significance of this visualization is that it can help us understant most frequestly done activities of user in comparison to other users.

    Finding from this visualization is that user 'aaa' is less intersted to use Favorite action where user 'bbb' is using it frequently. Both user 'bbb' and 'ccc' have not used DownVote. It seems they have found correct answers only.
